race_results/
=============
Computed race-result images, written automatically when a race is scored
(in the CLI or the web UI). Named:

    NNNN_YYYYMMDD_HHMMSS_<race name>.png      (NNNN = race number)

Only race results live here. User-generated reports and queries go in ../outputs/.
(The sample_race_*.png files are reference renders.)
